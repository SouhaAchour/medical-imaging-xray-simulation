# Medical Imaging X-Ray Simulation

## Project Overview
This project simulates X-ray imaging using Python.

The goal is to understand how imaging systems work and how data is processed in medical imaging.

## Objectives
- Simulate X-ray image formation
- Process imaging data
- Analyze results

## Technologies
- Python
- NumPy

## Files
- Partie1.py → main simulation
- Partie2.py → processing and analysis
- data_ct.npz → dataset
